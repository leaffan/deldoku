# Copilot Project Instructions

## Language Rules
- All explanations, reasoning steps, and chat responses should be written in **German**.
- All code comments must be written in **English**.
- All identifiers (variable names, function names, class names) must be in **English**.
- All console output, log messages, and strings inside the code must be in **English**.

## Style Rules
- Prefer clear, concise code.
- Avoid unnecessary abstractions unless explicitly requested.
- When multiple solutions are possible, choose the simplest readable one.

## Interaction Rules
- When the user asks a question, respond in German.
- When generating code, follow the language rules above without exceptions.
